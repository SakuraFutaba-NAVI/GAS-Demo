# GAS-Demo
UE5.7 GAS框架求职demo
基于 Unreal Engine 纯 C++ 扩展 GameplayAbilitySystem 实现的第三人称战斗 Demo，完整覆盖能力系统、属性系统、连击功能、敌人 AI、道具系统、UI 交互等。

##技术栈

引擎：Unreal Engine 5.7
开发语言：C++ 17
核心系统：GameplayAbilitySystem (GAS)、增强输入系统（Enhanced Input）
UI 系统：Unreal Motion Graphics (UMG)
网络：UE 原生网络同步、服务端权威模型
AI：行为树、AI 控制器
